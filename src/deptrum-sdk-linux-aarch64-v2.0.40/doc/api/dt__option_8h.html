kDtOptionStofDepthCropRoiStruct 

STOF depth cropping configuration (Y-direction only) uint8_t[4] = {x, y, width, height}.

  • Only cropping in Y direction is supported.
  • x must be 0.
  • width is fixed to 248.
  • y represents crop_up.
  • height = 180 - crop_up - crop_down. crop_up + crop_down < 170 crop_up + crop_down must be even
    Note
    For Nebula280 series, the ROI for MTOF and STOF must be set to the same values; cropping STOF or MTOF individually is not supported.
kDtOptionMtofDepthCropRoiStruct 

MTOF depth cropping configuration (Y-direction only) uint8_t[4] = {x, y, width, height}.

  • Only cropping in Y direction is supported.
  • x must be 0.
  • width is fixed to 248.
  • y represents crop_up.
  • height = 180 - crop_up - crop_down. crop_up + crop_down < 170 crop_up + crop_down must be even
    Note
    For Nebula280 series, the ROI for MTOF and STOF must be set to the same values; cropping STOF or MTOF individually is not supported.
