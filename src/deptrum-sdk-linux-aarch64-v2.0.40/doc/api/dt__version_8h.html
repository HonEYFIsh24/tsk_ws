DT_EXPORT int dt_get_version ( )

Retrieves the composite SDK version number.

Returns
int Composite version value combining major, minor and patch components.
Note
The version format is: (major * 10000000) + (minor * 10000) + patch.

DT_EXPORT const char* dt_get_version_str ( )

Retrieves the SDK version as a null-terminated string. The format is "major.minor.patch-stage". For example, "1.2.3-alpha". The returned string has static storage duration and should not be freed.

Returns
const char* Null-terminated string representing the SDK version.
