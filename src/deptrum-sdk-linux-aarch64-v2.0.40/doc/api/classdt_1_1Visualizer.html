Detailed Description

Default colormap table, 128 mapping values in total

When using the color mapping table, three adjustment parameters need to be settled, which are range_min, range_mid, range_max in millimeter (mm). In order to make the foreground image have more colorful color changes, the mapping function will map the last 100 color scales to range_min-range_mid and the first 28 color scales to range_mid-range_max.
