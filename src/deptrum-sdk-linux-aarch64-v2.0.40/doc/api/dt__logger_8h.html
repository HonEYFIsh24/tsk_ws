Detailed Description

Log management interface for configuring global logging behavior.

Provides functions to control log output destinations and severity thresholds.

DT_EXPORT void dt_set_logger_severity ( dt_log_severity  severity,
dt_error **  error 
)

Sets the global minimum severity threshold for all log outputs.

This setting affects console, file, and callback outputs simultaneously. Messages below the specified severity level will be discarded globally.

Parameters
[in]severityMinimum severity level to enforce globally.
[out]errorPointer to receive error details. Initialized to NULL before call, populated with error object on failure. Caller must release memory.
